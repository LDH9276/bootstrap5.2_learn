부트스트랩 버튼과 버튼 그룹

  • class = 'btn'
  • class = 'btn btn-dark'
  • class = 'btn btn-light'
  • class = 'btn btn-default'
  • class = 'btn btn-primary'
  • class = 'btn btn-success'
  • class = 'btn btn-info'
  • class = 'btn btn-wraning'
  • class = 'btn btn-danger'
  • class = 'btn btn-link'

btn 클래스는 button, input, a 태그 모두 적용이 가능하다.

btn 클래스의 외각선 적용은 btn-outline-으로 적용이 가능하다.

버튼 크기 지정하기 .btn-lg, .btn-sm

a요소 비활성화

a 요소에 비활성화 할때는 href 속성을 제거하고 disabled 속성을 넣어준다.

Primary Primary

버튼 블록

버튼을 블록 요소처럼 사용하고 싶을때는 .btn-block 클래스를 사용한다.

버튼 그룹

버튼을 그룹으로 묶어서 사용하고 싶을때는 .btn-group 클래스를 사용한다.
